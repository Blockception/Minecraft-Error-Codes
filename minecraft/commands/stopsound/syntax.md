# stopsound

## stopsound - Nr: 0

Stops all playing sounds on the given players.

```mcfunction
stopsound <player: selector> [sound: sound]
```

|Parameter|Type|Required|Other|
|:---|:---|:---|:---|
|`stopsound`|keyword|true||
|`player`|selector|true|Only players allowed<br/>|
|`sound`|sound|false||
