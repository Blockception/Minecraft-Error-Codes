# titleraw

## titleraw - Nr: 0

Sets the title.

```mcfunction
titleraw <player: selector> title <json raw text: jsonRawText>
```

|Parameter|Type|Required|Other|
|:---|:---|:---|:---|
|`titleraw`|keyword|true||
|`player`|selector|true|Only players allowed<br/>|
|`title`|keyword|true||
|`json raw text`|jsonRawText|true||
